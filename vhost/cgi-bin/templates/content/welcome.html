Labyrinth is a Website Management System, written in Perl and suitable for deployment on traditional Linux and Windows web server environments. This site consists of the project resources used to allow you to build on the codebase for Labyrinth, and create your own website.

This site together with the associated packages available from CPAN form the basis of a Demo website, that provides the system files, HTML templates and SQL files used by Labyrinth. These files can then be adapted and extended to create the website for your project. For further information please see the Labyrinth website.
